## 🚀 Overview

**Amazon Video Downloader** is a powerful Chrome extension that allows you to download product images and videos directly from Amazon product pages.

Designed for **e-commerce sellers, dropshippers, and marketers**, this tool helps you quickly collect high-quality media assets without manual effort.

---

## ✨ Features

- 📸 Download **main images, gallery images, and SKU images**
- 🎥 Download **product videos** (if available)
- ⚡ One-click bulk download
- 📦 Save files in organized formats
- 💻 Simple and lightweight – no complex setup required

## 🧑‍💻 How to Use

1. Install the extension from the Chrome Web Store  
2. Open any Amazon product page  
3. Click the extension icon in your browser  
4. View all available images and videos  
5. Click to download individually or in bulk
